Sistema de Agendamento Online — CRM Agendamentos

Instituição: Centro Universitário Carlos Drummond de Andrade — Tatuapé, SP
Período: Fevereiro 2026 – Maio 2026

Sobre o Projeto

Desenvolvimento de plataforma web completa para gerenciamento de agendamentos online, com banco de dados relacional integrado. O sistema visa substituir processos manuais por uma solução digital ágil, organizada e escalável — com foco na experiência do usuário e integridade dos dados.

Realizações Principais

• Dashboard com KPIs em tempo real (receita do dia, agendamentos, novos clientes, ticket médio) + gráfico anual de faturamento e distribuição de status por período

• Sistema completo de agendamentos com calendário visual mensal (heatmap de ocupação), timeline diária por profissional e validação de conflito de horário em tempo real

• Módulo financeiro com 5 períodos de análise (diário, semanal, mensal, multi-mensal, anual), gráficos dinâmicos via Chart.js 4.4.0 e exportação CSV

• Chat interno em tempo real via Supabase Realtime + gestão de clientes com avatar determinístico, badges de atividade (Ativo/Recente/Inativo) e badge VIP customizável

• Controle de acesso com 3 roles (admin / atendente / cliente), Row Level Security (RLS) em 11 tabelas PostgreSQL e timeout de sessão automático de 60 minutos

Tecnologias Utilizadas

Frontend: HTML5, CSS3, JavaScript ES6+ (ES Modules nativos — sem frameworks)

Backend/BD: Supabase v2 (PostgreSQL + RLS + Realtime + Auth)

Gráficos: Chart.js 4.4.0

Ícones / Fontes: Font Awesome 6.5 + Google Fonts (Inter)

Controle de Versão: Git e GitHub

Hospedagem: GitHub Pages com deploy automatizado

Gestor de TI — Política de Segurança da Informação (Estudo de Caso)

Instituição: Centro Universitário Carlos Drummond de Andrade — Tatuapé, SP
Período: Fevereiro 2024 – Junho 2024

Sobre o Projeto

Elaboração de uma Política de Segurança da Informação (PSI) completa para o Mercado Assaí como estudo de caso acadêmico. O projeto cobriu análise de riscos, gestão de ativos, controles de segurança e planos de resposta a incidentes — baseado nas melhores práticas internacionais.

Componentes da PSI

• Inventário e classificação de ativos de informação (hardware, software, dados)

• Análise qualitativa e quantitativa de riscos com metodologia OCTAVE Allegro

• Controles técnicos, administrativos e físicos baseados na ISO 27001/27002

• Matriz de riscos com análise de impacto e likelihood

• Plano de Continuidade de Negócios (PCN) e Plano de Resposta a Incidentes (PRI)

• Políticas de acesso, backup, criptografia e treinamento de usuários

Frameworks e Normas Aplicadas

Normas: ISO 27001, ISO 27002

Metodologia de Riscos: OCTAVE Allegro

Framework: NIST Cybersecurity Framework

Conformidade: LGPD (Lei Geral de Proteção de Dados)
